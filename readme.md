# Vendor Invoice Management System

## Project Overview

The Vendor Invoice Management System is developed using SAP CAP (Cloud Application Programming Model) and SAP Fiori Elements.

The application allows users to:
- Manage Vendors
- Manage Invoices
- View Invoice Items
- View Attachments
- Track Approval History
- Submit, Approve, and Reject invoices

---

## Technologies Used

- SAP CAP
- CDS
- OData V4
- SAP Fiori Elements
- SAPUI5
- SQLite

## Run the Project

Install dependencies

```
npm install
```

Start the project

```
cds watch
```

Application URL

```
http://localhost:4004/vendor.invoice.invoiceui
```
